# 5일만에 끝내는 클래식 음악사 — 김태용

- **번호**: 1130
- **날짜**: 2018-10-11
- **카테고리**: 예술/대중문화>음악
- **블로그**: [Link](https://m.blog.naver.com/naraeun33/221375154832)

---

서양 음악사 수업을 들은 것 같은 기분 (사실 교양시간에 들은 적 없음)

내용을 시대 별로 구분해서 구성한 것 같은데, 음악사 자체가 계속 연결되고 걸쳐져서 나오는 거라서 그런지, 구성이 이해의 흐름이랑 잘 맞지 않는 부분도 많아서, 정리하면서 조금 어려웠다.

현대 음악 설명 후 그냥 끝나버려서,(뒤에 저자와의 인터뷰가 있었지만) 뭔가 정리된 결론이 없는 것이 아쉬웠다.

그래도 이거 읽어서 그런지 클래식에 대해서 조금은 안다고 해도 될 것 같은 기분이 들었다.

이전에 관련 책 읽으면서 유명한 음악가들 출생-사망 연도 정리해놓은 것 첨부   
음악가들 연대 궁금할 때는 찾아보게 됨

---
#### 연대
고대 (그리스, 로마) - BC 6세기~ 4세기 - 여흥 음악   
중세 (로마) - 4세기~1450년 - 종교 음악   
르네상스 (네덜란드) - 1450~1600년 - 세속 음악   
바로크 (베네치아) - 1600~1750년 - 조성 음악 (비발디, 바흐)   
고전 (독일어권) - 1750~1810년 - 절대 음악 (모차르트, 베토벤)   
낭만 (독일) - 1810~1910년 - 표제 음악(슈만, 파가니니, 쇼팽, 리스트 / 인상주의 - 라벨, 드뷔시)   
현대 (다국적) - 1910~ - 무조 음악   

#### 고대
- 고대 로마 성가(cantus romanes): 악기가 개입하지 않은 무반주 단선율

#### 중세
- 그레고리오 성가(gregorian chant)
: 최초의 클래식 음악, 6-7세기, 그레고리오 1세 교황이 성가 정리
: 무반주 단선율, 음역의 폭이 좁음, 악보가 실존하지 않고 즉흥적인 노래, 남자만 노래하는 것을 원칙으로 함

- 네우마(neuma): 음악을 유지시키기 위한 첫 시도, 음의 높낮이 표시

- 계명창
: 웃(ut)-레(re)-미(mi)-파(fa)-솔(sol)-라(la) 6개 음절
: 17세기에 ut은 도(do)로 바뀜, 시(si)는 1600년대 이후 사용하기 시작함

#### 르네상스
- 샹송(chanson): 프랑스어를 가사로 하는 세속노래, 뱅슈와가 샹송을 부상시킴

- 돌림노래 -> 카논 -> 모방기법 -> 대위법(counterpoint)
: 네덜란드 음악의 대표적 특징, 시간차를 두고 앞선 선율을 다음 선율이 똑같이 따라감
: 바로크 시대의 바흐 --> 대위법의 진가

-기악음악: 공식석상이 아닌 사교모임에서 사용
: 기악의 연주기법 발전에 즉흥성 중요

- 토카타(toccata): 즉흥성을 지닌 오르간 장르, 건반위에서 즉흥연주를 하는 것

#### 바로크
- 절대주의적 정치상황에 얽매여서 음악가들은 좋은 대접을 받지 못하고 고용주와 고용인의 관계였음
- 비발디, 바흐, 헨델, 텔레만(공개 연주회 활성화) / 음악 외: 셰익스피어, 세르반테스, 루벤스, 렘브란트

- 대비
: 주선율의 확실한 위치 선점, 이를 떠받치는 하위 선율들의 생동감 있는 움직임
: 오페라(opera)와 협주곡(concerto)를 탄생시킴

- 기악 음악의 시대(sonata): 성악을 압도할만큼 대등한 위치에 올라선 기악

- 모노디 양식의 노래 (monody, 단선율)
: 피아노 반주가 있는 독창 노래, 독주나 독창의 연주를 돕는 반주 개념이 등장
: 낮은 음역대의 악기(오르간, 첼로, 바순)가 지속적으로 낮은 음역의 반주 역할

-조성 체계의 정착
: 장조(major)와 단조(minor)
: 장 필립 라모 - 으뜸화음(tonic) / 버금딸림화음(subdominant) / 딸림화음(dominant) 3화음을 조성 음악의 중심으로 삼음

- 화음과 화성
: 화음 - 높이가 다른 둘 이상의 음악이 동시에 울리는 것, 중심음이라는 하나의 축이 있어야 함
: 화성 - 화음들이 연속적으로 진행하는 것

- 감정이론: 작품 속에 감정을 담아내고자 함
: 곡의 주제를 잡는 것, 작품을 구상해서 스케치를 하는 일, 작곡하면서 장식을 만들어 넣는 것

- 음악어법
1) 바소 세쿠엔테 - 연속되는 베이스, 가장 아래 성부 움직임, 오르간 반주
2) 기악에서 관용적 어법 - 악기 음색에 따라 작곡하는 방식, 악기마다 차별적인 작곡법 등장
3) 음악어법의 양면성 - 리듬의 힘과 표현의 자유

- 건반악기
: 오르간 - 건반악기 가운데 줄이 없는 악기
: 쳄발로 - 건반을 누르면 깃털로 만든 채가 줄을 뜯어서 소리가 나는 악기, 바로크 대표 악기, 계속 저음 악기
: 클라비코드 - 지금 피아노의 전신이 되는 악기, 금속 장치가 줄을 때려서 소리가 나는 악기

- 협주곡(concerto): 바로크에서는 콘체르타토(concertato)라고 부름

- 오페라(opera)의 태동: 오페라의 첫 시작 부분 서곡(overture)

- 모음곡: 여러개의 춤곡을 한데 모아놓은 것

#### 고전
- 비엔나 고전: 모차르트, 하이든, 베토벤
- 단순 명료함, 국제화, 계몽주의
- 개개의 국가가 지닌 지역적 개성보다 어디서 누가 들어도 똑같이 즐길 수 있는 보편적 음악 추구

- 전 고전주의
: 로코코 양식 - 세련된 장식음이 운용
: 갈랑(galant) - 귀족과 궁정을 중심으로 일어난 문화적 활동, 처음 들릴 때 귀가 즐거워야 함
: 감정과다 양식(질풍노도) - 하나의 감정만을 드러내는 것이 아니라 한 악장안에서도 여러 다른 주제를 담아 다양한 감정을 보일 수 있게 하는 것

- 오케스트라의 표준화
: 정확한 악기의 지정과 인원이 배치, 오케스트라의 모습에 걸맞는 음악이 교향곡(symphony)

#### 낭만
- 비현실성, 감정적 효과를 극대화시킬 수 있는 모든 방법을 추구
- 예술 후원체계가 무너지고, 기득권에 귀속된 삶에서 벗어나 자유로운 상상력을 바탕으로 한 예술 작업 실현
- 슈만, 파가니니(바이올린), 쇼팽, 리스트 / 음악 외 괴테, 리히터, 위고, 하인리히

- 표제 음악(program music)
: 시, 소설, 신화를 소재로 삼아 작곡한 기악음악 (반대개념은 절대음악)
: 의도된 음악, 묘사가 아닌 표현에 중점을 둠

- 개량된 피아노의 페달(pedal) 기법

- 리트(lied): 시적인 가사와 음악이 통합된 독창과 피아노를 위한 성악작품

- 후기 낭만주의
: 인상주의: 인상이나 감정을 구체적으로 묘사하기 보다는 그 분위기를 불러일으킴 (드뷔시, 라벨 /음악 외: 세잔, 마네, 모네, 드가, 르누아르)
: 사실주의 오페라(푸치니)
: 바그너, 말러, 슈트라우스

-민족주의
:유럽 음악을 주도했던 독일, 프랑스, 이탈리아 이외의 국가들의 음악
: 러시아 (다르고미슈스키, 무소르크스키, 림스키코르사코프)

- 비민족주의: 차이코프스키, 루빈스타인, 스크라빈

#### 현대
- 초현실주의, 추상주의, 조성/리듬/형식을 규격하지 않은 비논리적 예술
- 다양성의 시대, 획일화된 양식을 버리고 자율적 선택으로 곡을 쓰게 됨
- 민족음악, 미래주의
- 스트라빈스키, 쇼스타코비치

- 음렬주의
: 조성을 배격한 자유로운 무조음악
: 12음 기법 - 모든 음이 동등한 자격, 모두가 으뜸음이 될 수 있음

- 신고전주의
: 고전 시대의 악기편성을 모방하는 소편성의 실내악
: 표제음악보다 절대음악 구조를 따름
: 라벨/ 스트라빈스키, 사티

#### 그 외
오페라
: 최초의 완본 오페라 - 에우리디체
: 레치타티보 - 말과 노래의 중간 형태로 연기, 음악적 낭송의 강조, 자유로운 리듬, 단순한 선율 고수
: 아리아 - 레치타티보보다 규칙적인 리듬, 유려한 선율 및 장식을 지닌 모노디 양식
오라토리오 (Oratorio)
: 종교적 주제에 재미를 위한 극적인 묘미가 감도는 종교적인 오페라, 해설자를 포함하고 있음
칸타타(cantata)
: 노래되는 곡, 장르로는 오페라와 오라토리오가 비슷, 형식면에서는 칸타타가 오라토리오와 비슷
: 실내에서 소규모 청중을 대상으로 연주

앙상블(ensemble): 2인 이상의 중창 또는 중주를 가리키는 실내음악, 다성부의 음악
스케르초(scherzo): 해학곡, 소나타, 교향곡, 현악4중주의 제 3악장에 쓰임, 템포가 빠른 3박자의 격렬한 리듬과 기분의 급격한 변화가 특징

Kochel-Verzeichnis(K/KV), 쾨헬 목록
: 루트비히 폰 쾨헬에 의해 처음 정리된 모차르트 모든 음악 작품 목록
: 이름을 줄여 작품 번호 앞에 K나 KV를 붙임

D(Deutsch)
: 슈베르트의 작품을 정리한 오토 에리히 도이치의 이름을 따서 붙인 약자
: 슈베르트 작품을 연대기적인 순서로 작품 번호 매김

WoO (Werke ohne Opuszahl)
: 작품번호가 없는 작품
: 주로 베토벤의 작품에 쓰임, 베토벤 작품 중 출판이 되지 않은 것, 사후에 발견된 곡
