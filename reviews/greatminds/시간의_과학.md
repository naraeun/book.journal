2025년 1월 14일-17일 방영되었고 4부작이다.

인간에게 시간은 기억과 생각과 예상, 과거를 떠올리며 미래를 예상하는 끊임없는 과정이고 이게 결국은 인간의 삶과 우리가 어떻게 보는지라는 관점이라는 사실과 연결해서 뻗어가는 과정이 넘 좋았다.    
과학의 위대함은 인류가 다 함께 성장하고 세상을 다르게 바라보는 관점을 제공해주면서 세상에 대해 배워가는 거고, 이를 쌓아온 과거의 과학자들을 존경하는 마음을 이야기해주는 것도 좋았다.    

'시간은 흐르지 않는다'에서 이야기한 내용을 강의로 구성했고 기대 많이 했는데, 기대했던 것보다 훨씬 재밌게 들었다. 글로 잘 표현하는 사람들은 말도 잘하신다. 필력이 괜히 나오는게 아니었다.

---

### 시간은 위에 있을 때 더 빨리 흐른다
- 아래쪽 시간이 느린 이유는 지구 질량, 중력이 있어서 시간이 느려짐
- 질량 큰 물체 주위에는 시간이 느려짐

### 우주 전체를 포괄하는 현재는 존재하지 않는다
- 지금 보는게 지금이라고 착각하지만 나에게 빛이 오는 시간이 걸리기 때문에 과거의 모습
- 현재를 잘 정의하는 방법은 범위를 내 주변으로 좁히는 것, 사건이 내 주변에서 벌어지면 빛이 이동하는 시간이 짧으니 눈에 보이는 걸 현재로 할 수 있음

### 뉴턴/맥스웰/아인슈타인 물리학 이론 방정식은 과거와 미래를 구분하지 않는다
- 과거와 미래가 다른 이유를 알려주지 않는다
- 한 방향으로 일어난 사건은 역방향으로도 일어날 수 있음

### 열역학으로 시간의 방향성을 설명할 수 있을 것 같았지만, 명확히 말해주지 않는다
- 열이 발생하는 곳엔 시간의 방향성이 존재
- 온도는 분자가 움직이는 속도에 불과하고 열 운동은 개별 분자 활동을 알지 못한다

### “우리가 어떻게 보는지”에 달린 것
- 우주는 그저 존재할 뿐인데, 우리의 관점이 시간이 흐른다고 생각하니깐 그렇게 인식한 것
- 어떤 세부사항을 지우고 보는지에 따라 우리의 관점이 잡힌다
- 인간의 관점에 따라 시간이 흐른다고 믿는다.

### 인간에게 시간이란 기억과 예상
- 과거를 떠올리며 미래를 예상하는 끊임없는 과정
- 현재는 한순간이 아니고, 과거와 미래에 동시에 존재하는 것
- 과거가 고정되어 있고 미래는 열려 있다. 그래서 과거에서 미래로 흐르는 것처럼 느낀다

### 시간의 흐름이란 우리가 생각하는 존재로서 시간을 느끼는 방식을 표현한 것
- 감정과 욕망을 통해서 시간을 느낌
- 시간이 흐른다는 건 인간이 풍부하게 느끼는 것인데, 이것을 자연에 그대로 대입하면 안된다

### 시공간을 양자의 관점에서 이해하는 양자 중력 이론
- 실험, 증거가 아직 없고 증명되지 않아서 가설
- 양자 중력에서 시간의 존재를 고려하지 않는다

### 양자 중력에서의 시간
- 시간은 연속되지 않고 조각나있다
    - 물리량은 연속적이지 않고 입자성을 가진다
    - 10-44초, 플랑크 시간, 제일 작은 시간 단위
    - 시간은 점프하면서 끊어진 단위로 이어진다
- 시간은 중첩되어 있다
    - 시간도 양자적으로 중첩되어 있다. 두 사건의 시간은 유일하지 않다
- 시간은 존재하지 않는다
    - 시공간 방정식 휠러-디윗 방정식에 시간 변수가 없음

### 시간이 없는 세상을 상상하기 어려우나, 시간은 다층적 개념
- 시간을 정의하는 유일한 개념이 없음
- 감정으로 느끼는 시간, 열역학의 시간
- 뉴턴: 시간은 존재하나 방향성은 없음
- 아인슈타인: 각자 시간에 따라 돌아감
- 양자중력: 시간은 존재하지 않음
